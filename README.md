<h2>Leipzschwimm</h2>

Web scraper that takes the water quality data of the Leipzig region swimming lakes from the official source (https://www.gesunde.sachsen.de/badegewaesser.html#EINSTUFUNG) to map it in an interactive Folium map.

<i>Note</i>: Right now (feb 2026) all the data is outdated because the sampling takes place only during the summer months.

Check it before swimming ust clicking on the Map Preview :)
